
Ciao i comandi di movimento sono i seguenti:
1) nord  -> se vuoi spostarti in avanti
2) sud   -> se vuoi spostarti indietro
3) est   -> se vuoi spostarti a destra
4) ovest -> se vuoi spostarti a sinistra

Comandi per la gestione della partita:
1) salva                   -> se vuoi salvare la partita
2) carica o carica partita -> se vuoi caricare la partita
3) nuova o nuova partita   -> se vuoi iniziare una nuova partita perdendo i salvataggi
4) aiuto                   -> per visualizzare questa schermata

Comandi per la storia:
1) prendi chiave                            -> per prendere la chiave
2) frammento mappa o prendi frammento mappa -> per raccogliere il frammento di mappa
3) sfonda porta                             -> per sfondare la porta
4) esamina stanza                           -> per esaminare la stanza
5) visualizza attributi                     -> per visualizzare gli attributi
6) visualizza inventario                    -> per visualizzare l'inventario
7) visualizza mappa			    -> per visualizzare i frammenti di mappa posseduti
8) apri botola                              -> per aprire la botola
